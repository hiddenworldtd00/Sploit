# HACKER_TCHAD_SPLOITE

**Développé par HiddenWorld Communauté Tchadienne**

Un framework éducatif complet pour comprendre et pratiquer la cybersécurité offensive de manière éthique et légale. Ce script est conçu comme un simulateur pédagogique qui reproduit les concepts et workflows de Metasploit Framework dans un environnement contrôlé.

## 🎯 Objectif

Comprendre les mécanismes de :
- Reconnaissance et scan de réseaux
- Exploitation de vulnérabilités (simulation)
- Post-exploitation et maintien d'accès
- Élévation de privilèges
- Création de payloads
- Techniques d'évasion

## ⚠️ Avertissement Légal

**Ce script est STRICTEMENT réservé à un usage éducatif et éthique :**
- ✅ Apprentissage de la cybersécurité dans un labo personnel
- ✅ Préparation aux certifications (OSCP, CEH, eJPT)
- ✅ Tests de pénétration autorisés par écrit
- ✅ CTF (Capture The Flag) et challenges légaux

**INTERDICTIONS ABSOLUES :**
- ❌ Utiliser sur des systèmes sans autorisation explicite
- ❌ Toute activité malveillante ou criminelle
- ❌ Attaquer des infrastructures publiques ou privées sans accord

## 📋 Prérequis

- Python 3.8+
- Linux recommandé (Kali Linux, Parrot OS)
- Droits root pour certaines fonctionnalités

## 🚀 Installation

```bash
# Cloner le repository
git clone https://github.com/hiddenworld/HACKER_TCHAD_SPLOITE.git

# Installer les dépendances
pip install -r requirements_sploite.txt

# Lancer le framework
python HACKER_TCHAD_SPLOITE.py
```

## 📖 Structure du Menu

### 1. Reconnaissance
- Scan de ports TCP/UDP
- Détection de services et versions
- Scan de vulnérabilités basique
- OS Fingerprinting

### 2. Exploitation
- Simulateur d'exploits (éducatif)
- Création de payloads
- Gestion des sessions

### 3. Post-Exploitation
- Collecte d'informations système
- Élévation de privilèges (simulation)
- Extraction de données
- Persistance

### 4. Payloads
- Reverse shells (simulation)
- Bind shells
- Meterpreter-like (simulation)
- Stagers

### 5. Evasion
- Techniques d'obfuscation
- Anti-virus evasion (concepts)
- Encodage de payloads

### 6. Sniffing & MITM
- ARP Spoofing (éducatif)
- Capture de paquets
- Analyse de trafic

### 7. Brute Force
- Attaques sur SSH, FTP, HTTP
- Dictionnaires personnalisés
- Threads configurables

### 8. Web Hacking
- Scan SQL Injection
- Scan XSS
- Scan LFI/RFI
- Découverte de répertoires

### 9. WiFi Hacking
- Scan des réseaux
- Découverte des clients
- Concepts de crack WPA/WEP

### 10. Rapports
- Génération de rapports HTML
- Export des résultats
- Recommandations
